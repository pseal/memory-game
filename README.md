# Interactive Memory Match Game

A sleek, responsive, web-based memory match game built from scratch. This project demonstrates interactive frontend logic, dynamic DOM manipulation, and state tracking wrapped in a clean, modern, and distraction-free user interface.

**Live Demo:**:https://memory-game-tau-coral.vercel.app/

# Features

- **Interactive Game Loop:** Dynamic grid generation with card-flipping mechanics, match validation, and mismatch resets.
- **State & Score Tracking:** Real-time tracking of player moves, match accuracy, and game completion states.
- **Visual Feedback:** Smooth, CSS-driven visual transitions and animations providing immediate interactive feedback to the user.
- **Responsive Layout:** A mobile-first design philosophy ensuring a seamless user experience across mobile, tablet, and desktop viewports.

# Tech Stack

- **Logic & Interactions:** Vanilla JavaScript (ES6+)
- **Structure:** Semantic HTML5
- **Styling:** Modern CSS3 (Flexbox/Grid, Transitions)
